## AI Research Lab이란?

**AI-ResearchLab**은 AI/ML 트렌드를 체계적으로 팔로업하고 팀의 인사이트로 큐레이션하는 연구 팀입니다.

### 우리가 하는 일

- 📡 **자동 수집**: arXiv, Hacker News, GitHub Trending, Hugging Face에서 매일 최신 동향 수집
- 📊 **팀 큐레이션**: 수집된 데이터를 바탕으로 매주 핵심 트렌드 선별 + 팀 인사이트 추가
- 📄 **논문 리뷰**: 주목받는 논문을 깊게 분석 (격주 팀원 로테이션)
- 🛠️ **직접 구현**: 트렌드 기술을 미니 프로젝트로 구현하고 결과 공유

### 레포지토리 구성

| 레포 | 역할 |
|------|------|
| [weekly-trends](https://github.com/AI-ResearchLab/weekly-trends) | 자동 수집 파이프라인 + 주간 리포트 |
| [paper-reviews](https://github.com/AI-ResearchLab/paper-reviews) | 논문 딥다이브 리뷰 |
| [implementations](https://github.com/AI-ResearchLab/implementations) | Hands-on 구현 프로젝트 |
| [awesome-ai](https://github.com/AI-ResearchLab/awesome-ai) | AI 툴/자료 큐레이션 목록 |
| [ai-researchlab.github.io](https://github.com/AI-ResearchLab/ai-researchlab.github.io) | 이 웹사이트 |

### 협업 방식

모든 콘텐츠는 **마크다운 파일**로 관리됩니다.  
팀원이 `.md` 파일을 PR로 올리면 GitHub Actions가 자동으로 빌드하고 배포합니다.

1. 이슈 등록 → 브랜치 생성 → 마크다운 작성 → PR → 리뷰 → 머지 → 자동 배포

자세한 기여 가이드는 [CONTRIBUTING.md](https://github.com/AI-ResearchLab/ai-researchlab.github.io/blob/main/CONTRIBUTING.md)를 참고하세요.
